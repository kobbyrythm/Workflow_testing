What could the site do?

This site is an open-data platform, and it makes everyone easily accessible to the data saved.

· Step 1: Go to the Datasets page.

· Step 2: You can use the searching box on Datasets page for filtering data.

· Step 3: The highlighting box on Data page helps you to locate the address of your keywords within thousands of words immidiately.

· Step 4: If you want to check the source page, you can find it under the Resource Ducuments according to the page number.

· Step 5: Any other confusions about the content of data, please contact the maintainer at the bottom of the data page.

How to upload one new page?

For avoiding useless data or other uncertain trouble, only administrators or the one with permission can upload new data.

If you are administrator, please go to the Github page and check the README file.

If you are not administrator, please go to the Help page and contact the administrator for the permission.
